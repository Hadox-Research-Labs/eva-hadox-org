# Security Contact

## Canonical Contact Path

Private security reports for this project should be sent to:

- `eva@ciencias.unam.mx`

This is the current primary security disclosure contact for the project.

## Reporting Guidance

When reporting a vulnerability, include:

- affected component or file;
- reproduction steps;
- realistic impact;
- whether the issue requires authenticated access, deployment exposure, or local access;
- any proposed mitigation, if known.

## Public Disclosure Rule

Do not disclose vulnerabilities publicly until maintainers confirm mitigation guidance or a fix is available.

## Canonical Repository Note

The canonical repository remains the Gitea project at `git.hadox.org`, but private security reports should go directly to the contact address above rather than public issue threads.

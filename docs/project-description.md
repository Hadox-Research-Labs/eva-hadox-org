# Project Description

This file provides reusable repository description text for public metadata in Gitea and GitHub.

## One-Line Description

Collaborative research platform for the historical study of breast cancer across archival, medical, and computational corpora.

## Short Description

EVA Breast Cancer Archive Platform is an open research infrastructure project for studying how breast cancer was named, described, classified, and treated before 1900 through historical sources, curated vocabularies, and computational corpus analysis.

## Medium Description

EVA Breast Cancer Archive Platform is a collaborative platform at the intersection of history of medicine, philosophy of classification, digital humanities, and computer science. It helps researchers build and explore a historical corpus on breast cancer through source metadata, curated lexical variants, mention detection, co-occurrence analysis, contextual similarity, and collaborative document ingestion.

## Long Description

EVA Breast Cancer Archive Platform is an interdisciplinary open research platform designed to support the historical study of breast cancer before 1900. The project combines archival and bibliographic curation, lexicon building, document ingestion, textual mention detection, co-occurrence analysis, and contextual retrieval in a single research environment. It is intended for work across history of medicine, philosophy, digital humanities, and computational text analysis, and serves both as a scholarly research instrument and as an open technical base for collaborative development.
